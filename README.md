# random-color.js

## Random-color.js is a NPM module that randomly generates a random color and outputs it in the terminal.
Here is an example.

```js
const { printDark } = require("@thekoneko/random-color.js");

printDark("Hello!");
```
### Its very simple, so here are the commands it has.
```js
printDark("Hey");

printBright("There");

printStyle("Person");
```
